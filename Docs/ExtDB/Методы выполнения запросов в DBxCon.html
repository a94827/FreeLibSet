Комментарии к таблице

В колонке «Первичный ключ» указано требование на наличие первичного ключа в таблице. Возможные значения:

Первичный ключ

Пояснение

-

Первичный ключ может быть любым или вообще отсутствовать.

Если используются ссылочные поля, то таблица, с которой выполняется объединение JOIN, должна иметь подходящий первичный ключ. В текущей реализации ссылочные поля могут иметь только тип Int32.

Int32

Таблица должна иметь первичный ключ, состоящий из одного поля типа Int32 (идентификатора записи)

любой тип, включая составной

Первичный ключ должен быть в таблице, но может быть произвольным



В колонке «Ссылочные поля» указано, можно ли использовать ссылочные поля с точками. Если у метода есть несколько перегрузок, то указаны значения для самой «продвинутой» перегрузки метода. Упрощенные версии метода могут не поддерживать ссылки.

Возможные значения: да, нет. Указано, если для некоторые параметров можно использовать поля с точками, а для других нет.
