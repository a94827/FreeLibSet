245

21.10.2021-29.10.2021

1. Изменения пространств имен

2. Классы FileAssociations и FileAssociationItem перенесены из ExtForms.

3. Свойство DepValue<T>.IsConnected (v.244 п.2)

4. Свойства EditableDateTimeFormatter.ContainsDate и ContainsTime, методы ToNValue (v.233 п.1).

5. Пространство имен FreeLibSet.UICore:

- Перечисление UIValidateState

- Классы UIValidator и UIValidatorList (перенесено из RI)

- Класс UITools. Методы TextToLines() и LinesToText(), ShiftDateRange() и ShiftDateRangeYear(). Методы GetIncrementedValue() (заглушки)

6. Удаленный интерфейс:

- Свойства HasXxxExProperty заменены на XxxPropertyConnected

- Вместо перечисления CanBeEmptyMode (v.229 п.3) используется UIValidateState

- Свойства Value типа Nullable<DateTime> переименованы в NValue (FirstValue, LastValue для диапазонов). Новые свойства Value используют тип данных DateTime.

- Классы DateBox и InputDateDialog заменены на DateTimeBox и InputDateTimeDialog(). Новое свойство Kind.

- Свойства DateOrRangeBox.FirstDate/Ex, LastDate/Ex

- Переименование DateInputDialog.Value/NValue.

- В диалогах ввода значений переименование MinValue/MaxValue в Minimum/Maximum. Для числовых типов используются nullable-значения

- Убрано свойство TextInputDialog.IsPassword. Предполагается создание классов PasswordTextBox и PasswordInputDialog



- Переименование ComboTextInputDialog в TextComboInputDialog. Свойство Items задается в конструкторе.

- Свойства TextInputDialog, TextComboInputDialog, MultiLineTextInputDialog.Value переименованы в Text

- MultiLineTextInputDialog.WriteChanges()/ReadChanges() передают свойство Lines, а не Text

- Свойство InfoLabel.Lines. Свойство Text является вспомогательным

7. Методы DataTools.DecimalPlacesToNumberFormat() и DecimalPlacesToNumberFormat() перенесены в FormatStringTools

См. также версию 1.1 и описание перехода на версию 2.0
