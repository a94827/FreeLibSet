24.11.2021-25.11.2021

1. Класс InputGridDataDialog (v.232 п.1) переименован в InputDataGridDialog. Изменена реализация свойства Columns. Используются классы InputDataGridColumn и InputDataGridColumns. Параметры столбца задаются как свойства, а не вызовом методов.

3. Конструктор CheckBox устанавливает начальное значение NChecked=false, а не null (v.301 п.12). Имеет значение, если для переключателя не задано свойство Name.

4. Внутреннее поле DepInput<T>._CheckValueArgs (v.301 п.5) помечено атрибутом [NonSerialized], чтобы не возникало исключение сериализации при наличии обработчика CheckValue.

5. Неуправляемые свойства RI.Control.Enabled (v.167 п.1) и RI.TextBox.ReadOnly (v.229 п.3) не работали.

6. Свойства ReadOnly и ReadOnlyEx у следующих элементов удаленного интерфейса:

- DateTimeBox

- BaseNumEditBox<T>

- CsvCodesComboBox

- FolderBrowserTextBox

- FileTextBox

21.10.2021-22.11.2021

Изменена реализация метода SetValueChanged(): после установки Source=null текущее значение не меняется. Раньше устанавливалось значение default<T>.

6. Класс DepValueObject<T> (v.91 п.1) переименован в DepOutput<T>. Конструктор требует аргумент value.

7.Классы DepExprX<...> наследуют DepValue<T>, а не DepOutput<T>

8. Методы DepTools.InRange/Ex<T>() (v.244 п.2)

9. Свойства EditableDateTimeFormatter.ContainsDate и ContainsTime, методы ToNValue (v.233 п.1).

10. Пространство имен FreeLibSet.UICore:

- Перечисление UIValidateState, интерфейс IUIValidableObject (был EFPValidator в ExtForms), класс UISimpleValidableObject (был EFPSimpleValidator), делегат UIValidatingEventHandler.

- Классы UIValidator и UIValidatorList (перенесено из RI, v.244 п.3)

- Класс UITools. Методы TextToLines() и LinesToText(), ShiftDateRange() и ShiftDateRangeYear().

- Поддержка прокрутки значений стрелочками вверх/вниз. Интефрейсы IUpDownHandler<T> и IMinMaxSource<T>. Абстрактный класс IncrementUpDownHandler<T>, реализации Int/Single/Double/DecimalUpDownHandler. Класс NumArrayUpDownHandler<T> для прокрутки на основе фиксированного списка значений.

11. Статические методы CfgPart.IsValidName() и ValidateName() (v.24 п.1). Изменены абстрактные методы, чтобы выполнялась проверка корректности имени. Допускаются только буквы, цифры и символы ".", "-" и "_". Первым символом должна быть буква.

12. Удаленный интерфейс (v.59 п.11):

- Свойства HasXxxExProperty заменены на InternalXxxPropertyConnected

- Свойства Value типа Nullable<DateTime> переименованы в NValue (FirstValue, LastValue для диапазонов). Новые свойства Value используют тип данных DateTime.

- Классы DateBox и InputDateDialog заменены на DateTimeBox и InputDateTimeDialog(). Новое свойство Kind.

- Свойства DateOrRangeBox.FirstDate/Ex, LastDate/Ex (v.175 п.1)

- Переименование DateTimeInputDialog.Value/NValue (v.71 п.3).

- Свойства IsNotEmptyEx.

- В диалогах ввода значений и диапазонов переименование MinValue/MaxValue в Minimum/Maximum. Для числовых типов используются nullable-значения. Новые свойства Format, UpDownHandler и Increment (в IntEditBox убрано свойство ShowUpDown)

- Классы PasswordBox и PasswordInputDialog

- Убрано свойство TextInputDialog.IsPassword.

- Убраны свойства Error/WarningRegExPattern/Message (v.229 п.3)

- Переименование ComboTextInputDialog в TextComboInputDialog. Свойство Items задается в конструкторе.

- Свойства TextInputDialog, TextComboInputDialog, MultiLineTextInputDialog.Value переименованы в Text

- MultiLineTextInputDialog.WriteChanges()/ReadChanges() передают свойство Lines, а не Text

- Свойство InfoLabel.Lines. Свойство Text является вспомогательным

- Управляемые свойства FolderBrowserTextBox, FileTextBox, PathEx и IsNotEmptyEx.

- CheckBox поддерживает свойства CanBeEmpty, CanBeEmptyMode, IsNotEmptyEx и NChecked/Ex вместо ThreeState и CheckState.

- Свойство CsvCodesComboBox.UnknownCodeSeverity, CodeValidators и ValidatingCodeEx.

- В диалогах ввода значения и диапазона значений - валидаторы, как в управляющих элементах (свойства Validators, v.244 п.3) и управляемые свойства для построения валидаторов.

- Проверяется корректность свойства RIItem.Name
