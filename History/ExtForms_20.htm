304

26.11.2021-30.11.2021

2. Значением по умолчанию свойства InputDataGridColumn.Align (v.303 п.1) для числовых типов данных является HorizontalAlignment.Right, а не Left

3. Провайдер поля ввода EFPCsvCodesTextBox. Поддержка удаленного интерфейса (CsvCodesTextBox, ExtTools, v.304 п.5)

4. Изменения в столбцах для TreeViewAdv. Вместо классов NodeNumericUpDown, NodeDecimalTextBox, NodeIntegerTextBox и внутренний класс NumericTextBox используются классы NodeIntEditBox, NodeSingleEditBox, NodeDoubleEditBox, и NodeDecimaEditBox, использующие новый управляющий элемент NumEditBoxBase (v.301 п.12)

303

24.11.2021-25.11.2021

1. Класс InputGridDataDialog (v.276 п.2) переименован в InputDataGridDialog. Изменена реализация свойства Columns. Используются классы InputDataGridColumn и InputDataGridColumns. Параметры столбца задаются как свойства, а не вызовом методов. Аналогично FreeLibSet.RI.InputGridDataDialog (ExtTools, v.303 п.1)

2. Еще одна попытка избавиться от неперехватываемого исключения ObjectDisposedException, которое иногда появляется из-за всплывающих подсказок в EFPFormProvider (v.287 п.1). Объект ToolTip удаляется в обработчике Form.Disposed.

3. Неуправляемые свойства RI.Control.Enabled и RI.TextBox.ReadOnly не работали (ExtTools, v.303 п.5).

4. Свойства ReadOnly у некоторых элементов удаленного интерфейса (ExtTools, v.303 п.6)

302

23.11.2021

1. Исправлена ошибка в EFPAppRI.RadioGroupItem (v.94 п.6). Начальное состояние инициализировалось неправильно.

2. Установка свойства DateTimeBox.NValue (v.301 п.5) неправильно отбрасывала компонент времени

3. Событие EFPAppRICreators.BeforeCreate (v.106 п.5).

4. Исправлены ошибки в NumEditBox<T>. Генерируется событие TextChanged, которое используется в EFPNumEditBox<T> для вызова Validate(), вместо события ValueChanged.

301

21.10.2021-22.11.2021

1. Изменения пространств имен

2. Убран метод DebugTools.LogoutException() (v.85 п.1)

3. Класс Twain (v.41 п.1) переименован в FreeLibSet.Win32.Twain.TwainObject

4. Перечисление EFPValidateState заменено на UIValidateState (ExtTools, v.301 п.10)

5. Свойство EFPControlBase.Validators.

6. Управляющий элемент DateBox (v.1 п.1) заменен на DateTimeBox, а DateInputDialog (v.76 п.4) - на DateTimeInputDialog. Новые свойства Kind и Formatter. Свойство Value переименовано в NValue. Новое свойство Value типа DateTime без значения null.

7. Класс EFPDateBox (v.17 п.2) заменен на EFPDateTimeBox. Классы EFPDateTimeBox и EFPDateTumePicker выведены из общего класса EFPDateTimeControl<T>, который поддерживает свойства NValue, Value, NTime и Time. Также поддерживается комплект свойств DisabledXxx и XxxEx.

8. Свойство EFPMonthCalendarSingleDay.SyncValue пропускает устанавливаемое значение null, а не выбрасывает исключение.

9. Свойства DateRangeBox (v.1 п.2), EFPDateRangeBox.FirstDate и LastDate переименованы в First и Last

10. Убраны управляемые свойства EFPDateOrRangeBox.ValueEx и IsNotEmptyEx (v.197 п.2). Свойство Value переименовано в DateRange. Новые свойства FirstDate, LastDate, FirstDateEx и LastDateEx.

11. Свойства CanBeEmptyMode вместо WarningIfEmpty. Убрано CanBeEmptyModeEx, MinimumEx, MaximumEx, MaxLengthEx, Error/WarningRegExPattern/Message

12. Новые управляющие элементы IntEditBox, SingleEditBox, DoubleEditBox и DecimalEditBox с поддержкой произвольного форматирования и инкремента. Провайдеры EFPxxxEditBox. Убраны элементы NumEditBox (v.1 п.3) и ExtNumericUpDown (v.46 п.6) и классы провайдеров EFPNumEditBox, EFPExtNumericUpDown и EFPNumericUpDown.

13. Диалоги ввода значений Int/Single/Double/DecimalInputBox (v.76 п.4) и RangeBox используют новые элементы и свойства Format, FormatProvider, UpDownHandler и Increment.

14. В диалогах ввода значения и диапазона значений - валидаторы, как в управляющих элементах (свойства Validators) и управляемые свойства для построения валидаторов. Убраны события Validating.

15. EFPCheckBox (v.17 п.2) поддерживает свойства CanBeEmpty, CanBeEmptyMode, IsNotEmptyEx и NChecked/Ex вместо ThreeState и CheckState.

16. Убраны методы EFPApp.InputBox() и IntInputBox().

17. Переименование ComboTextInputDialog (v.76 п.4) в TextComboInputDialog. Добавлено свойство Lines.

18. Методы DateRangeBox.ShiftDateRange() (v.13 п.3) и EFPDateRangeBoxCommandItems.ShiftDateRangeYear() (v.229 п.4) перенесены в UITools.

19. Переименование свойств RadioGroupBox.ImageIndexes (v.242 п.4) в ImageIndices.

20. Свойства EFPFolderBrowserButton, EFPFileDialogButton, EFPWindowsExplorerButton.Path, PathEx и IsNotEmptyEx.

21. Методы WinFormsTools.IsImageShrinkNeeded() (v.198 п.1) перенесены в ImagingTools.

22. Вспомогательные классы TreeViewAdv (v.92 п.2) переименованы, некоторые перенесены в пространство имен FreeLibSet.Controls.TreeViewAdvInternal.

23. Методы EFPAppClipboard.SetTextMatrix(), WinFormsTools.GetTextMatrix(), GetTextMatrixCsv(), GetTextMatrixText(), SetTextMatrix() (v.163 п.6). Использование TabTextConvert и CsvTextConvert вместо методов DataTools.

24. UserComboBoxBase обрабатывает нажатие клавиши <F8> для нажатия кнопки "Очистить". Очистка с помощью клавиши <Пробел> сохранено для UserSelComboBox.

25. UserComboBoxBase (v.46 п.4) при изменении свойства Enabled с false на true обновляет цвета основного управляющего элемента.

26. Событие EFPCsvCodesComboBoxBase.CodeValidating (v.258 п.2). Свойства CodeValidators и ValidatingCodeEx (в основном, для удаленного интерфейса). Класс больше не является абстрактным.

27. Свойство EFPCsvCodesComboBox.UnknownCodeSeverity (v.258 п.2).
