29.10.2025-07.11.2025

1. Скрытая форма в EFPAppTrayIcon (v.206 п.9) перехватывает событие FormClosing и не разрешает закрывать приложение, даже если нет ни одной открытой формы. Без этого свойство EFPAppTrayIcon.UseHiddenForm было практически бесполезным

2. EFPFormProvider запоминает (v.279 п.1) положение (а не только размер) окон с Form.BorderStyle=FixedToolWindow и SizeableToolWindow

3. EFPFormProvider в обработчике Form.FormDisposed вызывает EFPApp.ExitAsync(), если больше нет открытых форм и tray-значка,

4. Свойство EFPApp.AppWasInit переименовано в AppHasBeenInit

5. EFPApp.ShowWindowsExplorer() (v.39 п.1) может принимать путь к файлу вместо каталога. Если используется explorer.exe (v.210 п.7), то он вызывается с аргументом «/select,путь-к-файлу» для позиционирования на файл

6. В окне «Загруженные программные модули» (v.67 п.3) - команда локального меню «Открыть папку с файлом»

- Метод ReadData()

- События Load, BeforeReadData, AfterReadData, BeforeWriteData, AfterWriteData. Свойство UseDataWriting. Событие Writing переименовано в Applying.

5. EFPIntEditBox переименован в EFPInt32EditBox (v.366 п.1)

367

02.10.2025-08.10.2025

1. Свойство IEFPControl.DefaultButton (v.351 п.46). Конструкторы EFPFolderBrowserButton и EFPFileDialogButton устанавливают его. Отменено 19.11.2025, неудобно.

2. EFPAppCommandItems.CreateStdCommand() добавляет комбинации клавиш к стандартным командам SaveAs (Ctrl+Shift+S), CopyHyperlink (v.366 п.9) (Ctrl+Shift+H)

3. Свойство EFPCommandItems.ShortCuts (v.17 п.6) для поддержки нескольких комбинаций клавиш на команду. Internal-свойство AlternativeShortCuts больше не используется.

4. EFPToolBarBase.Add() не добавляет родительские команды для команд с флагом EFPCommandItemUsage.ToolBarAux (v.355 п.20) в выпадающее меню панели инструментов

5. В локальном меню табличного (EFPDataGridViewCommandItems) и иерархического (EFPTreeViewCommandItemsBase, v.167 п.3) просмотров выделены подменю «Правка», «Буфер обмена» и «Поиск»

6. Для команд «Предыдущий/следующий период времени» фильтра табличного просмотра назначены комбинации клавиш «Ctrl+[<]» и «Ctrl+[>]», как в DateRangeBox, Старые комбинации «Alt+Стрелка вверх» и «Alt+Стрелка вниз» сохранены для совместимости

7. EFPMenuOutHandler (v.348 п.31) создает подменю «Файл», в котором размещает команды «Печать», «Параметры страницы», «Предварительный просмотр» и «Экспорт в файл»

8. В табличном просмотре EFPDataGridView (v.54 п.1) – команды локального меню «Перейти вниз/вверх к строке с другим значением» (комбинации клавиш Alt+PageUp и Alt+PageDown). Для EFPDataTreeView не реализовано.

9. Перечислимое свойство IEFPDataView, EFPDataGridViewCommandItems, EFPTreeViewAdvCommandItems.EnterKeyMode, расширяющее EnterAsOk. Старое свойство сохранено для совместимости.

10. Свойство DateRangeCrossGridFilter.Nullable (v.366 п.8) для поддержки режимов NULL и NOT NULL (ExtTools. v.367 п.1)
