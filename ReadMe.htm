Совместимость

Язык: C# 2.0

Библиотеки совместимы с .Net Framework 2 и более новыми версиями

 

Если используется Microsoft Visual Studio, то требуется версия 2005 или новее.

Файлы проектов *.csproj пригодны для включения в решения для Visual Studio 2005, 2008, 2010, 2012 и 2013

Лицензия

Указанные выше библиотеки относятся к свободному программному обеспечению и могут распространяться в виде исходных кодов или в компилированном виде.

В том числе, разрешается модификация и использование в коммерческом ПО

См. лицензионное соглашение.

Используемое стороннее ПО

Некоторая часть исходных текстов библиотек заимствует свободное программное обеспечение, разработанное другими авторами. Соответствующие ссылки и оригинальные авторские права указаны в исходных текстах

 

Кроме этого, используются динамические ссылки на следующие сторонние библиотеки, расположенные в подкаталоге Others
